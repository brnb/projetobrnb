    Mais coisas importantes sobre as listas:

    • As listas ordenadas podem começar com um número diferente do 1, pra isso, basta criar um atributo "start" para a "ol" e incluir a posição de início;
    • Listas ordenadas também podem começar de trás pra frente, pra isso é usado o atributo "reversed";
    • Dá pra criar listas dentro de listas:
      • Assim!
      • Serve
      • Pra
      • Irritar
      • o Usuário
    • O atributo "type" serve pra mostrar o tipo de número, numa lista ordenada... Pode ser romanos maiúscula ou minúscula... Mas isso também dá pra fazer com o CSS.
